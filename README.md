# VisionGuard — Real-Time Wrong-Way Vehicle Detection

AI-powered road safety system using YOLOv8 and OpenCV.

## What it does
Detects vehicles travelling the wrong way in traffic footage
and raises an instant visual + audio alert in real time.

## Tech Stack
- Python 3.10+
- YOLOv8 (Ultralytics)
- OpenCV
- SORT Tracker

## Setup
pip install ultralytics opencv-python numpy sort-tracker playsound

## Run
python main.py

## Accuracy
~90% on standard highway footage
